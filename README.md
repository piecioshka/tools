# tools

🛠 Stack of Tools

## Preview 🎉

<https://piecioshka.github.io/tools/>

## Features

- ✅ Remove Text Formatting - clean pasted text and keep it in LocalStorage
- ✅ Binary Converter - encode text to binary and decode binary to text
- ✅ Slugify - generate URL-friendly slugs with Unicode normalization and Polish character support
- ✅ Base64 Converter - encode text to Base64 and decode Base64 to text
- ✅ Proportions - calculate proportional values and display simplified ratios
- ✅ Date Diff - calculate the number of days between two dates
